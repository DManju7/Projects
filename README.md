# 🎮 Gesture & Voice Game Control Using Python

> Control games with the power of your hands and voice — no keyboard, no controller, just natural interaction.

## 🚀 Project Vision

Imagine playing your favorite game without touching a keyboard or controller.

This project combines **Computer Vision**, **Speech Recognition**, and **Human-Computer Interaction (HCI)** to create a touchless gaming experience. Using a webcam and microphone, players can perform hand gestures or speak commands that are instantly translated into game actions.

## 🎮 How It Works

### Gesture Control

The webcam captures hand movements and MediaPipe identifies 21 hand landmarks. Specific finger combinations are mapped to game actions.

### Voice Control

The microphone captures spoken commands. Vosk converts speech into text, and the system executes the corresponding game action instantly.
